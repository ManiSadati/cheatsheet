# engineering-learning

This repository stores engineering notes and small experiments focused on:

- compilers
- MLIR
- performance engineering
- LLM systems
- optimization

## Structure

```
engineering-learning
│
├── compilers/
├── mlir/
├── performance/
├── llm-systems/
├── experiments/
├── templates/
│   ├── note-template.md
│   └── experiment-template.md
│
├── README.md
├── learning-log.md
└── AGENTS.md
```

## Conventions

- Keep notes in Markdown files under topic directories.
- Keep experiments isolated in subfolders under `experiments/`.
